# das-home

Home Assistant Dashboard mit SAP UI5 Web Components.

## Features

- 24+ Karten-Typen mit Auto-Discovery
- Widget-Wizard fuer einfache Konfiguration
- HACS Custom Card Bridge
- Area-Views mit Tab-Navigation
- Popup-System fuer Detail-Ansichten
- Dark/Light Theme

## Installation

Dieses Add-on ueber den Add-on Store installieren oder die Repository-URL manuell hinzufuegen:

```
https://github.com/conuti-das/das-home
```

## Konfiguration

| Option | Standard | Beschreibung |
|--------|----------|-------------|
| `debug` | `false` | Debug-Logging aktivieren |
